# PowerGuard

Приложение для Android, которое автоматически отключает функции устройства при бездействии (экран выключен). Помогает экономить заряд батареи без ручного вмешательства.

## Возможности

| Функция | Управление |
|---|---|
| Авто-поворот | Прямое — отключается автоматически |
| Яркость экрана | Прямое — снижается до минимума |
| Авто-яркость | Прямое — переключается в ручной режим |
| Громкость медиа | Прямое — отключается звук |
| Режим звонка | Прямое — вибрация или тишина (с DND) |
| Bluetooth | Прямое на API ≤32 / уведомление на API 33+ |
| Wi-Fi | Уведомление со ссылкой на настройки |
| NFC | Уведомление со ссылкой на настройки |

### Для каждой функции настраивается:
- Тайм-аут бездействия (1–120 минут)
- Предупреждение за N минут до действия
- Исключения: при зарядке, в домашней Wi-Fi сети, в заданный период времени
- **Восстановление при разблокировке** — автоматически включает функцию обратно, когда пользователь разблокирует устройство (только для функций с прямым управлением)

### Дополнительно:
- Журнал событий с историей всех действий
- Пауза автоматизации на 1 час
- Запуск при загрузке устройства
- Интерфейс на русском и английском языках
- Поддержка тёмной темы

## Требования

- Android **10** (API 29) и выше
- Разрешения запрашиваются при первом запуске через экран «Главная»

## Сборка

```bash
git clone https://github.com/TheVolfy/powerguard.git
cd powerguard
./gradlew assembleDebug
```

APK будет в `app/build/outputs/apk/debug/app-debug.apk`.

## Стек технологий

- **Kotlin** 2.0.21
- **Jetpack Compose** + Material 3
- **Room** 2.6.1 — база данных журнала событий
- **DataStore** 1.1.1 — хранение настроек
- **Coroutines** 1.9.0
- **Navigation Compose** 2.8.4
- **Foreground Service** с динамической регистрацией BroadcastReceiver

## Архитектура

```
MVVM
├── UI Layer       — Compose screens + ViewModels
├── Domain Layer   — FeatureType, FeatureRule (модели)
├── Data Layer     — SettingsRepository (DataStore), EventLogRepository (Room)
└── Service Layer  — MonitoringService (Foreground), FeatureController
```

## Разрешения

| Разрешение | Назначение |
|---|---|
| `WRITE_SETTINGS` | Управление яркостью и авто-поворотом |
| `ACCESS_NOTIFICATION_POLICY` | Режим полной тишины (DND) |
| `POST_NOTIFICATIONS` | Уведомления (API 33+) |
| `BLUETOOTH_CONNECT` | Управление Bluetooth (API 31+) |
| `ACCESS_FINE_LOCATION` | Чтение SSID для исключения домашней сети |
| `RECEIVE_BOOT_COMPLETED` | Автозапуск после перезагрузки |
